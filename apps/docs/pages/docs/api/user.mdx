# User
<Callout type="info">
  This is the documentation for the latest version of Next Admin. If you are using an older version (`<5.0.0`), please refer to the [documentation](/v4/docs)
</Callout>

The `user` property allows you to define the user displayed in the sidebar menu.
Its an object that can take the following properties:

<OptionsTable
  options={[

      name: "data.name",

      type: "String",
      description: "the URL of the user's avatar displayed in the sidebar menu",
    },
    {
      name: "logoutUrl",
      type: "[RequestInfo, RequestInit?] | Function (server action)",
      description: <>either a request tuple (<code>[RequestInfo, RequestInit?]</code>) to fetch the logout API or a server action. This is required.</>,

  ]}
