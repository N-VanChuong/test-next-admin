# Customize the admin route name

<Callout type="info">
  This is the documentation for the latest version of Next Admin. If you are using an older version (`<5.0.0`), please refer to the [documentation](/v4/docs/route)
</Callout>

When setting up `next-admin` you can set the admin panel route name to whatever you want.
The admin panel route name is set by your folder name.

Examples:

`pages/admin/[[...nextadmin]].tsx` => `basePath` will be `/admin` (Page Router)

`pages/api/admin/[[...nextadmin]].ts` => `apiBasePath` will be `/api/admin` (API Page Router)

`pages/prisma/[[...nextadmin]]/page.tsx` => `basePath` will be `/admin` (App Router)

`pages/prisma/[[...nextadmin]]/route.ts` => `apiBasePath` will be `/api/admin` (API App Router)
